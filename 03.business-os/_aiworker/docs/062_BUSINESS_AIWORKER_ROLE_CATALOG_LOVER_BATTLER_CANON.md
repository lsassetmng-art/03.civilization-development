# BusinessOS AIWorker Role Catalog Add Lover / Battler Canon

## Purpose
Add two placement roles to BusinessOS AIWorker role catalog.

## Added roles
- Lover
- Battler

## Meaning
### Lover
擬似恋人系・ラバー系AIワーカー用の配置ロール。
LoVerSシリーズ、HD-R1A、MEGAMI NORNのFriend/Lover寄り利用で使用する。

### Battler
戦闘員・戦闘担当・特殊実行系AIワーカー用の配置ロール。
HD-R2 / HD-R2G / HD-R2T-0 などで使用する。

## Note
Butler は「執事」。
Battler は「戦闘員」。
今回の戦闘員系ロールには Battler を使う。

## Scope
This step only updates role catalog.
It does not assign roles to robot_pool.
