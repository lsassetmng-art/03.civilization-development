# AICompanyManager BusinessOS AIWorker Auth Token Client Canon

## Purpose
Provide shared browser helper for optional API token headers.

## Storage key
- aicm_business_aiworker_api_token

## Header
- X-AICM-AIWORKER-TOKEN

## Window override
- window.AICM_BUSINESS_AIWORKER_API_TOKEN

## Helper
- window.AICM_BUILD_AIWORKER_AUTH_HEADERS(extraHeaders)
