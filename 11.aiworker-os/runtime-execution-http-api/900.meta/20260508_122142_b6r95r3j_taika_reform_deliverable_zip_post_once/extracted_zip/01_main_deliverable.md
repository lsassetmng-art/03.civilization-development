# 大化の改新 詳細資料

## 1. 概要

大化の改新とは、7世紀半ばの日本で進められた政治改革の総称である。一般には、645年の乙巳の変によって蘇我蝦夷・蘇我入鹿を中心とする蘇我氏本宗家が滅ぼされ、その後、中大兄皇子・中臣鎌足らが中心となって、天皇を中心とする中央集権国家の形成を進めた一連の改革を指す。

ただし、現在の歴史理解では「645年に一気に完成した単発の改革」というより、7世紀半ばから後半にかけて、唐の律令制度や東アジア情勢の影響を受けながら段階的に進んだ国家形成過程として捉えるのが自然である。

## 2. 時代背景

### 2.1 蘇我氏の台頭

6世紀から7世紀前半にかけて、蘇我氏は大王家との婚姻関係、仏教受容、渡来系技術・文化の活用などを背景に大きな政治的影響力を持った。蘇我馬子、蘇我蝦夷、蘇我入鹿の時代には、蘇我氏本宗家が朝廷内で非常に強い権力を握った。

### 2.2 豪族連合的な政治体制

大化の改新以前のヤマト政権は、天皇中心の完全な中央集権国家ではなく、有力豪族が土地・人民・部民を支配する豪族連合的な性格が強かった。各豪族は私有民や私有地を持ち、朝廷の支配は地域社会の隅々に直接及んでいたわけではなかった。

### 2.3 東アジア情勢の緊張

7世紀の東アジアでは、隋に続いて唐が強大な帝国として成立し、朝鮮半島では高句麗・百済・新羅が対立していた。日本列島の政権にとって、唐の制度や軍事力は強い衝撃であり、国防・外交・制度整備の面で国家体制を強化する必要が高まっていた。

## 3. 乙巳の変

### 3.1 発生年

乙巳の変は645年に起こった政変である。中大兄皇子と中臣鎌足らが、飛鳥板蓋宮で蘇我入鹿を暗殺し、蘇我蝦夷も自害したことで、蘇我氏本宗家は滅亡した。

### 3.2 中心人物

主な人物は以下の通りである。

- 中大兄皇子: 後の天智天皇。改革の中心人物。
- 中臣鎌足: 後の藤原鎌足。中大兄皇子の協力者で、藤原氏の祖。
- 蘇我入鹿: 蘇我氏本宗家の有力者。乙巳の変で暗殺された。
- 蘇我蝦夷: 入鹿の父。政変後に自害した。
- 皇極天皇: 乙巳の変当時の天皇。事件後に譲位した。
- 孝徳天皇: 乙巳の変後に即位し、大化の年号が定められた。

### 3.3 意義

乙巳の変は、蘇我氏本宗家の専横を打破し、天皇中心の政治体制へ移行する契機となった。ただし、この時点で律令国家が完成したわけではなく、改革の出発点として位置づけるのが適切である。

## 4. 大化の年号と新政権

乙巳の変後、皇極天皇は退位し、孝徳天皇が即位した。日本で初めての元号とされる「大化」が定められた。新政権では、中大兄皇子が皇太子として政治を主導し、中臣鎌足らが補佐した。

この新政権は、従来の豪族中心の政治から、天皇を中心とする官僚制国家へ移行しようとした点に特徴がある。

## 5. 改新の詔

646年に出されたとされる「改新の詔」は、大化の改新の基本方針を示すものとして知られる。『日本書紀』に記録されているが、後世の律令制度を反映して整理された可能性も指摘されているため、史料批判が必要である。

主な内容は以下の四つに整理される。

### 5.1 公地公民制

豪族が私的に支配していた土地や人民を廃止し、土地と人民を国家、つまり天皇の支配下に置くという考え方である。

意義:
- 豪族の私的支配を弱める。
- 国家が人民と土地を直接把握する。
- 律令国家の基礎を作る。

注意点:
- 実際に全国で直ちに実現したわけではない。
- 地域社会では旧来の豪族支配も一定程度残ったと考えられる。

### 5.2 地方行政制度の整備

国・郡・里などの地方行政単位を整え、中央から国司を派遣して地方を統治する方向が示された。

意義:
- 中央政府が地方を直接支配する仕組みを作る。
- 豪族の地域支配を国家機構に組み込む。
- 戸籍・税・軍事動員の基盤を整える。

### 5.3 戸籍・計帳と班田収授

人民を戸籍で把握し、一定の条件で田地を分配する仕組みが目指された。後の班田収授法につながる考え方である。

意義:
- 国家が人民を把握する。
- 課税対象を明確にする。
- 兵役や労役の基礎を作る。

### 5.4 税制の整備

租・庸・調など、律令制下の税体系につながる制度が整えられていった。

- 租: 田地に課される稲の税。
- 庸: 労役の代わりに納める布など。
- 調: 地方の特産物などを納める税。

意義:
- 国家財政を安定させる。
- 中央政府が地方資源を把握する。
- 官僚制・軍事・公共事業を支える。

## 6. 政治制度の変化

大化の改新以後、中央政府は官僚制を整え、天皇を中心とした国家機構を作ろうとした。のちの律令制度では、二官八省などの中央官制が整えられ、太政官を中心に行政が行われるようになった。

大化期そのものでは制度が未完成であったとしても、この時期に中央集権化の方向性が明確になったことは重要である。

## 7. 地方支配の変化

地方では、国司・郡司などの制度が整えられていった。国司は中央から派遣される官人であり、郡司には旧来の地方豪族が任命されることが多かった。

この仕組みにより、国家は地方豪族を完全に排除するのではなく、国家機構の中に取り込みながら地方支配を進めた。

## 8. 国防と外交への影響

大化の改新は内政改革であると同時に、東アジア情勢への対応でもあった。唐や新羅の動向、朝鮮半島情勢の緊迫化は、日本列島の政権に中央集権化と軍事体制整備を促した。

663年の白村江の戦いで日本は唐・新羅連合軍に敗北し、その後、防人・水城・山城などの防衛体制整備が進んだ。これは、大化の改新後の国家体制強化の流れと結びつく。

## 9. 律令国家への道

大化の改新は、すぐに完成した律令国家そのものではない。しかし、後の近江令、飛鳥浄御原令、大宝律令へとつながる国家形成の重要な出発点である。

特に大宝律令によって、中央官制、地方制度、戸籍、班田、税制などが制度として整備され、日本は本格的な律令国家へ移行した。

## 10. 歴史的意義

大化の改新の意義は以下のように整理できる。

1. 蘇我氏本宗家の打倒により、豪族連合的政治から天皇中心政治への転換が始まった。
2. 公地公民制の理念により、土地・人民を国家が把握する方向が示された。
3. 地方行政制度、戸籍、税制、軍事動員など、律令国家の基盤が整えられた。
4. 東アジアの国際環境に対応するため、中央集権化が進められた。
5. 後の大宝律令による律令国家形成への道筋を作った。

## 11. 史料上の注意点

大化の改新については、『日本書紀』の記述が中心資料となる。しかし『日本書紀』は8世紀に編纂された国家的歴史書であり、後の律令国家の立場から7世紀の改革を整理している可能性がある。

そのため、改新の詔に書かれた制度が646年にそのまま実施されたと見るのではなく、後世の制度を反映した叙述である可能性も考慮する必要がある。

## 12. まとめ

大化の改新は、645年の乙巳の変を契機として始まった政治改革であり、日本が豪族連合的な体制から天皇中心の中央集権国家へ向かう重要な転換点であった。公地公民、地方行政制度、戸籍、班田、税制などの理念は、後の律令国家形成に大きな影響を与えた。

ただし、改革は一度に完成したものではなく、7世紀後半から8世紀初頭にかけて段階的に整備されたと理解することが重要である。
